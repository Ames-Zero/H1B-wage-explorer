# H1B Wage Interactive Dashboard

An interactive Streamlit dashboard that visualizes H1B prevailing wage data across US geographical regions with filtering capabilities for job roles and wage levels.

## Features

- **Interactive Maps**: Visualize wages by state or county
- **Job Filtering**: Filter by 850+ job roles and 4 wage levels
- **Salary Classification**: See which wage level your salary falls into across different areas
- **Statistics**: View average, median, max, and min wages
- **Data Explorer**: Browse raw filtered data


## Data Source

All wage data is sourced from the **U.S. Department of Labor - Office of Foreign Labor Certification (OFLC)**.

Wage data files are publicly available at: https://flag.dol.gov/wage-data/wage-search

This application uses the 2025-26 wage year data.
